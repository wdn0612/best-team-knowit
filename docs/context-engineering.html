2 / Memory Management Architecture
👤
UserProfile
rnai-user-profile
称呼 用户的名字
情绪模式 长期情绪规律 + why + lastMentioned
兴趣 关注点 + 对用户的意义
目标 可完成的具体目标
价值观 持续的生活方向 + evidence（ACT 核心）
重要的人 name + relation
沟通偏好 回应风格期望
⏱ 兴趣/目标/情绪模式 30 天衰减 ☆ 价值观 60 天衰减
💓
EmotionTrajectory
rnai-emotion-trajectory
主导情绪 开心/焦虑/难过/平静/…
强度 1-5 级，5 最强烈
触发因素 导致情绪的具体事件
趋势分析 本周 vs 上周自动对比
⏱ 存储 14 天
✎ 注入 prompt 取最近 7 条
📝
ConversationSummary
rnai-conversation-summaries
摘要 一两句话概括对话核心
重要性 routine / notable / milestone
主题标签 ["工作压力", "人际关系"]
🕒 routine — 7 天后不注入
⭐ milestone — 永久保留
📦 最多存 20 条，注入取 3 + milestones
📅
LifeEvent
rnai-life-events
事件 有明确时间锚点的未来事件
情感倾向 excited / anxious / neutral
关心类型 before（事前鼓励）/ after（事后关心）
推送通知 到 checkInTime 自动触发本地推送
🚨 pending → checked_in → 7 天后清理
🚫 只提取未来 30 天内的事件
